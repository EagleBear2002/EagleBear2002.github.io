  2. a = 1 + 2

  1. 在 C++中，非 const 引用可以绑定到左值，而 const 引用可以绑定到左值和右值，但是没有什么可以绑定到非 const r 值。

    左值 右值

    const 引用 可绑定 可绑定
    右值引用 不可绑定 可绑定

符号 含义
[] Capture nothing
[&] Capture any referenced variable by reference
[=] Capture any referenced variable by making a copy
