技术分析入门

发表于 2025-03-04 更新于 2026-02-09 分类于 投资研究 阅读次数：
本文字数： 2.2k 阅读时长 ≈ 2 分钟

摘要

本文是技术分析入门文章，介绍了技术分析的原理，系统梳理了常用的技术指标，包括趋势类指标、震荡类指标、能量类指标和复合指标。

技术分析三大前提和四个分析要素

三大前提是技术分析的基础：

  1. 市场行为能够包容和消化一切；
    1. 股价的波动和涨跌是一切影响股价的因素的集中体现；
    2. 这些因素包括公司经营状况、行业甚至政治、心理等；
    3. 影响股价变动的根本原因是基本面因素；
    4. 没找到股价波动的原因很可能是还没发现相关因素；
  2. 价格以趋势的方式演变；
    1. 股价的趋势变化是技术分析的核心；
    2. 价格波动要么上涨趋势，要么下跌趋势，要么是横盘震荡；
    3. 趋势一旦确立，不会轻易改变或者终止；
    4. 通过技术分析工具找到市场趋势，然后顺势而为。
  3. 历史会重演；
    1. 各种“图标”给出的涨跌信号总会不断重演；
    2. 同样的技术分析信号，过去有效，将来一样会有效；
    3. 人性的贪婪和恐慌不变，历史将不断重演；
    4. “图标”中给出的技术信号不会完全雷同，但会非常相似；
    5. 中国股市中相邻的两个顶部或者相邻的两个底部，形态非常类似；相隔时间越长，形态差别越大；

四个分析要素：

  1. 价格
    1. 价格是我们研究技术分析的首要因素；
    2. 价格是指股票过去和当前的成交价，具体的还有开盘价、收盘价、最低价和最高价；
    3. 每一日内价格波动是市场上所有多空双方博弈的结果，它直接反映在当日收盘价上；
    4. 股价连续三天收盘价突破重要关口，才算是真的突破了；
  2. 成交量
    1. 成交量反应市场人气和动力的指标，哥价格之间存在一定的推理关系；
    2. 上涨需要放量，否则持久不了，下跌则不需要，可以缩量下跌（因为推动上涨需要真金白银，而少量抛售就能引发下跌）；
    3. 通常先看到成交量变化，然后再看到价格的波动；成交量信号提前于股价变化；
    4. 量价背离是指价格上涨但是成交量不增加甚至减少，是趋势转折极其重要的信号；
  3. 时间
    1. 时间是指时间周期；
    2. 牛市和熊市是存在周期性的交替；
    3. 股价存在周期性变化，因为股价反映经济，经济本身存在周期性；
    4. 研究时间的目的在于寻找市场的趋势转折点；
  4. 空间
    1. 空间就是股价上下波动的范围；
    2. 预测价格就是预测股价波动的空间范围，要精确预测很难；
    3. 以时间换空间是指股价上的空间和时间相互转化；

趋势类指标（判断方向）

移动平均线（MA）

  • 原理：计算过去 N 天的平均价格，反映平滑后的趋势。
  • 应用：
    • 短期（如 5/10 日）MA 向上穿越长期（如 30/60 日）MA 为金叉（买入信号）。
    • 反向为死叉（卖出信号）。
  • 适用：中长线趋势跟踪，如股票、商品期货。

指数移动平均线（EMA）

  • 原理：赋予近期数据更高权重，比 MA 更敏感。
  • 应用：与 MA 结合使用，减少滞后性。

布林带（Bollinger Bands）

  • 原理：中轨为 N 日均线，上下轨为 ±2σ 标准差，衡量波动性。
  • 应用：
    • 价格触及上轨后回落（卖出），下轨后反弹（买入）。
    • 收窄通道预示突破。
    • 适用：高波动市场（如加密货币、外汇）。

震荡类指标（判断超买超卖）

相对强弱指数（RSI）

  • 原理：比较上涨与下跌幅度，值域 0-100。
  • 应用：
    • RSI>70 表示买方力量强，为超买（卖出），<30 表示卖方力量强，为超卖（买入）。
  • 底背离（价格新低但 RSI 新高）预示反转。
  • 适用：短期交易（如股票日内、外汇）。

随机振荡器（Stochastic Oscillator）

  • 原理：%K 线（当前价与 N 周期极值的百分比）与%D 线（%K 的 3 日 MA）。
  • 应用：
    • %K 上穿%D 为买入，下穿为卖出。
    • 超买/超卖区域与趋势结合使用。
  • 适用：趋势明确的震荡市。

能量类指标（量价关系）

成交量（Volume）

  • 原理：单位时间交易量，验证价格走势的可靠性。
  • 应用：
    • 价格上涨伴随放量（趋势延续）。
    • 下跌无量（可能见底）。
  • 适用：所有市场，尤其低流动性品种（如小盘股）。

能量潮（OBV）

  • 原理：累计成交量，正值表示买力强，负值卖力强。
  • 应用：OBV 与价格背离时预警趋势反转。

复合指标（综合信号）

MACD（Moving Average Convergence Divergence）

  • 原理：12/26/9 日 EMA 的差值（DIF）及其 9 日 EMA（DEA）。
  • 应用：
    • DIF 上穿 DEA 为金叉（买入），下穿为死叉（卖出）。
    • 底背离（价格下跌但 DIF 上升）看涨。
  • 适用：中短期趋势，如股票、原油。

DMI（Directional Movement Index）

  • 原理：通过+DI/-DI 衡量上升/下降趋势强度，ADX（平均趋向指数）判断趋势强弱。
  • 应用：ADX>25 为趋势市，配合 DI 信号交易。

关键注意事项

  • 多指标协同：单一指标易滞后，需结合（如 MA+MACD+RSI）。
  • 参数调整：不同周期/市场需优化参数（如日线 vs.小时线）。
  • 过滤噪音：震荡市慎用趋势指标，趋势市忽略短期波动。
  • 风险管理：技术分析不保证盈利，需配合止损策略。
  • 示例场景：
    • 股票长线：60 日均线支撑 + MACD 金叉 + 成交量放大 → 加仓。
    • 外汇短线：RSI 超卖 + 布林带下轨支撑 + 1 小时 EMA 金叉 → 多头入场。
  • 掌握指标本质（反映市场心理与资金流动）比机械套用更重要。
