10-иҪҜд»¶дҪ“зі»з»“жһ„и®ҫи®ЎдёҺжһ„е»ә

EagleBear2002 2022-05-31 14:56:19
Categoriesпјҡ иҪҜд»¶е·ҘзЁӢдёҺи®Ўз®—II Tagsпјҡ

дҪ“зі»з»“жһ„и®ҫи®ЎиҝҮзЁӢ(з®Җз•ҘжӯҘйӘӨ) йҮҚиҰҒ

  1. еҲҶжһҗе…ій”®йңҖжұӮе’ҢйЎ№зӣ®зәҰжқҹпјҡеҲҶжһҗз”ЁдҫӢж–ҮжЎЈе’ҢйңҖжұӮи§„ж јиҜҙжҳҺд№Ұ(еҢ…еҗ«йңҖжұӮи§„ж је’ҢйЎ№зӣ®зәҰжқҹ)гҖӮжіЁж„Ҹж—ўиҰҒиҖғиҷ‘еҠҹиғҪжҖ§йңҖжұӮпјҢеҸҲиҰҒиҖғиҷ‘йқһеҠҹиғҪжҖ§йңҖжұӮпјҢз”ҡиҮіеҫҲеӨ§ж„Ҹд№үдёҠдҪ“зі»з»“жһ„и®ҫи®ЎжҳҜдёәдәҶж»Ўи¶ійқһеҠҹиғҪжҖ§йңҖжұӮгҖӮ
  2. йҖҡиҝҮйҖүжӢ©дҪ“зі»з»“жһ„йЈҺж јпјҡйҖүжӢ©еҲҶеұӮйЈҺж ј(дҝЎжҒҜзі»з»ҹгҖҒе№¶иЎҢејҖеҸ‘гҖҒйқһwebеә”з”Ё)пјҢиҝӣиЎҢиҜ„е®ЎгҖӮ
  3. иҝӣвҫҸиҪҜд»¶дҪ“зі»з»“жһ„йҖ»иҫ‘(жҠҪиұЎ)и®ҫи®Ўпјҡдә§з”ҹйҖ»иҫ‘еҢ…еӣҫ
  4. дҫқиө–йҖ»иҫ‘и®ҫи®ЎиҝӣиЎҢиҪҜд»¶дҪ“зі»з»“жһ„(е®һзҺ°)и®ҫи®Ўпјҡдә§з”ҹзү©зҗҶзұ»еӣҫ
  5. е®Ңе–„дҪ“зі»з»“жһ„и®ҫи®Ўпјҡе…ій”®зұ»еӣҫпјҢжҢҒд№…еҢ–ж•°жҚ®ж јејҸзҡ„е®ҡд№үзӯү
  6. ж·»еҠ жһ„д»¶жҺҘеҸЈпјҡеҢ…гҖҒйҮҚиҰҒж–Үд»¶зҡ„еҲӣе»әпјҢе®ҡд№үжҺҘеҸЈ
  7. иҝӯд»ЈиҝҮзЁӢ3-6
  1. жӯҘйӘӨ1-3жҳҜйҖ»иҫ‘и®ҫи®ЎпјҢжӯҘйӘӨ4-7жҳҜзү©зҗҶи®ҫи®Ў
  2. жң¬ж–ҮжҺҘдёӢжқҘзҡ„йғЁеҲҶе°ҶиҰҒжҢүз…§иҝҷдёӘжҖқи·ҜиҝӣиЎҢдёӢеҺ»

  1. е°ҶйңҖжұӮеҲҶй…ҚеҲ°еӯҗзі»з»ҹе’ҢжЁЎеқ—
    1. иҖғиҷ‘еҠҹиғҪзҡ„зӣёеҗҢжҖ§пјҡдёҚеҗҢд»»еҠЎпјҢдҪҶжҳҜзӣёеҗҢеҠҹиғҪ
    2. иҖғиҷ‘еҸҜеӨҚз”ЁжҖ§пјҡз»“жһ„гҖҒж•°жҚ®гҖҒиЎҢдёәзҡ„еҸҜеӨҚз”ЁжҖ§
  2. е®һи·өжЎҲдҫӢ

    2. дә§е“Ғи°ғж•ҙгҖҒе…Ҙеә“гҖҒеҮәеә“дёҺеә“еӯҳеҲҶжһҗ

    4. й”Җе”®зӯ–з•Ҙ

    1. й”Җе”®

    3. дјҡе‘ҳ

    2. йҖ»иҫ‘еұӮеҜ№иұЎиҝӣиЎҢдёҖдёҖдәӨдә’пјҢеҸӘиҰҒдә’зӣёеҜ№еә”зҡ„иҝӣиЎҢйҖҡзҹҘеҚіеҸҜгҖӮ
    3. иҝӣдёҖжӯҘе°ҶдёҡеҠЎеұӮзҡ„еҠҹиғҪиҝӣиЎҢеҲҮеүІеҲ’еҲҶ

дҪҝз”ЁйқһеҠҹиғҪжҖ§йңҖжұӮдёҺйЎ№зӣ®зәҰжқҹиҜ„д»·е’Ңж”№иҝӣеҲқе§Ӣи®ҫи®Ў

еҲқжӯҘи®ҫи®Ўзҡ„еҲҶжһҗ

  1. иғҪеӨҹж»Ўи¶ійЎ№зӣ®зәҰжқҹ:
    1. еҲҶеұӮйЈҺж јиғҪдҝғиҝӣе№¶иЎҢејҖеҸ‘,д»ҺвҪҪзј©зҹӯејҖеҸ‘ж—¶й—ҙ;
    2. еҲҶеұӮйЈҺж јеҸҜд»ҘдҪҝвҪӨ Java жҠҖжңҜ,вҪҪдёҚдҪҝвҪӨ Web жҠҖжңҜгҖӮ
  2. ж— жі•ж»Ўи¶іе®үе…ЁйңҖжұӮ(Security1~3)е’ҢзҪ‘з»ңеҲҶеёғзәҰжқҹ(IC2),жүҖд»ҘйңҖиҰҒж”№иҝӣ:
    1. дёәдҪҝе…¶ж»Ўи¶іе®үе…ЁйңҖжұӮ,еҸҜд»ҘеўһеҠ з”ЁжҲ·зҷ»еҪ•дёҺйӘҢиҜҒеҠҹиғҪ,еҸҜд»Ҙе»әз«Ӣдё“й—Ёзҡ„дёүдёӘжЁЎеқ— (PresentationгҖҒLogicгҖҒ Data),д№ҹеҸҜд»Ҙе°ҶиҜҘеҠҹиғҪе№¶вјҠвҪӨжҲ·з®ЎзҗҶеҠҹиғҪ,еҚідёә userui, user, userdata дёүдёӘжЁЎеқ—еўһеҠ ж–°зҡ„иҒҢиҙЈгҖӮ
    2. дёәж»Ўи¶ізҪ‘з»ңеҲҶеёғзәҰжқҹ,йңҖиҰҒе°ҶжЁЎеқ—еҲҶеёғеҲ°е®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁз»„жҲҗзҡ„зҪ‘з»ңдёҠгҖӮ
    3. еҸҜд»Ҙе°Ҷ Prensentation еұӮжЁЎеқ—йғЁзҪІеңЁе®ўжҲ·з«Ҝ,е°Ҷ Logic еұӮе’Ң Data еұӮзҡ„жЁЎеқ—йғЁзҪІеңЁжңҚеҠЎеҷЁз«ҜгҖӮ
    4. д№ҹеҸҜе°Ҷ Prensentation еұӮе’Ң Logic еұӮжЁЎеқ—йғЁзҪІеңЁе®ўжҲ·з«Ҝ,е°Ҷ Data еұӮзҡ„жЁЎеқ—йғЁзҪІеңЁжңҚеҠЎеҷЁз«ҜгҖӮ
    5. вјҖж—ҰзӣёйӮ»дёӨеұӮиў«йғЁзҪІеҲ°зҪ‘з»ңдёӨз«Ҝ,йӮЈд№Ҳе®ғд»¬д№Ӣй—ҙзҡ„дәӨдә’е°ұвҪҶжі•йҖҡиҝҮзЁӢеәҸи°ғвҪӨжқҘе®ҢжҲҗ,еҸҜд»ҘиҖғиҷ‘е°Ҷз®ҖеҚ•зҡ„зЁӢеәҸи°ғвҪӨиҪ¬еҢ–дёәиҝңзЁӢвҪ…жі•и°ғвҪӨ RMI(Remote Methods Invocation)жҲ–иҖ…HTTPгҖӮ
  3. иҝһй”Ғи¶…еёӮз®ЎзҗҶзі»з»ҹжңҖз»Ҳзҡ„иҪҜд»¶дҪ“зі»з»“жһ„йҖ»иҫ‘и®ҫи®ЎвҪ…жЎҲ

Package Design Principles еҢ…и®ҫи®ЎеҺҹеҲҷ

  1. йҮҚз”ЁеҸ‘еёғзӯүд»·еҺҹеҲҷ(REP):йҮҚз”Ёзҡ„зІ’еәҰе°ұжҳҜеҸ‘еёғзҡ„зІ’еәҰ
  2. е…ұеҗҢе°Ғй—ӯеҺҹеҲҷ(CCP):еҢ…дёӯжүҖжңүзұ»еҜ№дәҺеҗҢдёҖзұ»жҖ§иҙЁзҡ„еҸҳеҢ–еә”иҜҘжҳҜе…ұеҗҢе°Ғй—ӯзҡ„пјҢдёҖдёӘеҸҳеҢ–иӢҘеҜ№дёҖдёӘеҢ…дә§з”ҹеҪұе“ҚпјҢеҲҷеҜ№иҜҘеҢ…дёӯзҡ„жүҖжңүзұ»дә§з”ҹеҪұе“ҚпјҢиҖҢеҜ№дәҺе…¶д»–еҢ…дёҚйҖ жҲҗд»»дҪ•еҪұе“ҚгҖӮ
  3. е…ұеҗҢйҮҚз”ЁеҺҹзҗҶ(CRP):дёҖдёӘеҢ…дёӯзҡ„жүҖжңүзұ»еә”иҜҘжҳҜиғҪеӨҹе…ұеҗҢйҮҚз”Ёзҡ„гҖӮ
  4. ж— зҺҜдҫқиө–еҺҹеҲҷ(ADP):еңЁеҢ…зҡ„дҫқиө–е…ізі»еӣҫдёӯдёҚиғҪеӯҳеңЁзҺҜгҖӮ
  5. зЁіе®ҡдҫқиө–еҺҹеҲҷ(SDP):жңқзқҖзЁіе®ҡзҡ„ж–№еҗ‘иҝӣиЎҢдҫқиө–
  6. зЁіе®ҡжҠҪиұЎеҺҹеҲҷ(SAP):еҢ…зҡ„жҠҪиұЎзЁӢеәҰеә”иҜҘе’Ңе…¶зЁіе®ҡзЁӢеәҰдёҖиҮҙ
  7. еүҚдёүжқЎжҸҸиҝ°зҡ„жҳҜдҫқиө–жҖ§пјҢеҗҺдёүжқЎжҸҸиҝ°зҡ„жҳҜиҖҰеҗҲжҖ§

Common Closure Principle (CCP) е…ұеҗҢе°Ғй—ӯеҺҹеҲҷ

  1. дёҖиө·дҝ®ж”№зҡ„зұ»еә”иҜҘж”ҫеңЁдёҖиө·
  2. жңҖе°ҸеҢ–дҝ®ж”№еҜ№зЁӢеәҸе‘ҳзҡ„еҪұе“Қ
  3. еҪ“йңҖиҰҒжӣҙж”№ж—¶пјҢеҜ№зЁӢеәҸе‘ҳжңүеҲ©
  4. еҰӮжһңжӣҙж”№з”ұдәҺзј–иҜ‘е’Ңй“ҫжҺҘж—¶й—ҙд»ҘеҸҠйҮҚж–°йӘҢиҜҒиҖҢеҪұе“ҚдәҶе°ҪеҸҜиғҪе°‘зҡ„иҪҜд»¶еҢ…
  5. зӨәдҫӢ

    2. йқўеҗ‘еҸҜд»Ҙйў„жңҹзҡ„еҸҳжӣҙе°Ғй—ӯеҢ…
    3. е°Ҷжӣҙж”№йҷҗеҲ¶дёәеҮ дёӘиҪҜд»¶еҢ…

    5. еҮҸе°‘зЁӢеәҸе‘ҳзҡ„е·ҘдҪңйҮҸ
    6. еҸӘеҜ№еҸҜйў„жөӢзҡ„еҸҳжӣҙжңүдҪңз”ЁпјҢдёҚеҸҜйў„жөӢзҡ„еҸҳжӣҙдјҡдёәзі»з»ҹеёҰжқҘжһҒеӨ§зҡ„з ҙеқҸиғҪеҠӣпјҢе№¶дё”ж— жі•иҝӣиЎҢйў„жөӢгҖӮ

Common Reuse Principle (CRP) е…ұеҗҢйҮҚз”ЁеҺҹеҲҷ

  1. дёҖиө·иў«йҮҚз”Ёзҡ„еә”иҜҘеңЁдёҖиө·
  2. еҢ…еә”йҮҚзӮ№зӘҒеҮәпјҢз”ЁжҲ·еә”дҪҝз”ЁеҢ…дёӯзҡ„жүҖжңүзұ»
  3. жҖ»з»“
    1. ж №жҚ®еёёи§ҒйҮҚз”ЁеҜ№зұ»иҝӣиЎҢеҲҶз»„пјҡйҒҝе…Қз»ҷз”ЁжҲ·дёҚеҝ…иҰҒзҡ„дҫқиө–
    2. йҒөеҫӘCRPйҖҡеёёдјҡеҜјиҮҙиҪҜд»¶еҢ…жӢҶеҲҶпјҡиҺ·еҫ—жӣҙеӨҡпјҢжӣҙе°ҸпјҢжӣҙдё“жіЁзҡ„еҢ…
    3. еҮҸе°‘йҮҚж–°дҪҝз”ЁиҖ…зҡ„е·ҘдҪң

  1. CCPе’ҢCRPеҺҹеҲҷжҳҜдә’ж–Ҙзҡ„пјҢеҚідёҚиғҪеҗҢж—¶ж»Ўи¶іе®ғд»¬гҖӮ

  3. CCPиҮҙеҠӣдәҺдҪҝеҢ…иЈ…е°ҪеҸҜиғҪеӨ§пјҢиҖҢCRPеҲҷе°қиҜ•дҪҝеҢ…иЈ…еҫҲе°ҸгҖӮ
  4. еңЁйЎ№зӣ®зҡ„ж—©жңҹпјҢжһ¶жһ„еёҲеҸҜд»Ҙи®ҫзҪ®еҢ…з»“жһ„пјҢдҪҝCCPеҚ дё»еҜјең°дҪҚпјҢе№¶еҚҸеҠ©ејҖеҸ‘е’Ңз»ҙжҠӨгҖӮеҗҺжқҘпјҢйҡҸзқҖдҪ“зі»з»“жһ„зҡ„зЁіе®ҡпјҢжһ¶жһ„еёҲеҸҜд»ҘйҮҚжһ„зЁӢеәҸеҢ…з»“жһ„пјҢд»ҘдҪҝеӨ–йғЁйҮҚз”ЁзЁӢеәҸзҡ„CRPжңҖеӨ§еҢ–гҖӮ
  5. д№ҹе°ұжҳҜиҜҙеңЁиҪҜд»¶ејҖеҸ‘зҡ„дёҚеҗҢйҳ¶ж®өCCPе’ҢCRPзҡ„йҮҚи§ҶйҮҚз”ЁзЁӢеәҰж—¶дёҚеҗҢзҡ„пјҢиҰҒж №жҚ®йЎ№зӣ®иҝӣеұ•жқҘиҝӣиЎҢдёҚеҗҢзЁӢеәҰзҡ„еӯҰд№ гҖӮ

    4. еҮҸе°‘йҮҚж–°дҪҝз”ЁиҖ…зҡ„е·ҘдҪң
    5. е’Ңзӣёе…іиҒ”зҡ„зұ»дёҖиө·еҸ‘еёғпјҢиҖҢдёҚжҳҜеҚ•зӢ¬иҝӣиЎҢеҸ‘еёғ

    2. вҪҗеҰӮ,иҝһй”Ғи¶…еёӮзі»з»ҹдёӯеӯҳеңЁе®ўжҲ·з«Ҝе’ҢжңҚеҠЎеҷЁз«ҜгҖӮйӮЈд№ҲжҲ‘д»¬е°ұйңҖиҰҒеңЁе®ўжҲ·з«ҜйҖҡиҝҮ GUI з•ҢвҫҜеҸ‘еҮәжҢҮд»Ө,йҖҡиҝҮе®ўжҲ·з«Ҝзҡ„дёҡеҠЎйҖ»иҫ‘еӨ„зҗҶ,и®ҝй—®жңҚеҠЎеҷЁз«Ҝзҡ„ж•°жҚ®,еҜ№ж•°жҚ®иҝӣиЎҢдҝ®ж”№гҖӮеҸӘжңүиҝҷж ·жүҚиғҪиЎЁжҳҺеҪ“еүҚзҡ„дҪ“зі»з»“жһ„еҸҜд»Ҙиғңд»»еҠҹиғҪжҖ§йңҖжұӮгҖӮ
  2. еҜ№еҺҹеһӢзҡ„йқһеҠҹиғҪжҖ§жҢҮж ҮиҝӣвҫҸдј°з®—е’ҢйӘҢиҜҒгҖӮеҰӮжһңеҮә зҺ°дёҚз¬ҰеҗҲйқһеҠҹиғҪжҖ§йңҖжұӮе’ҢйЎ№зӣ®зәҰжқҹзҡ„жғ…еҶө,жҲ‘д»¬иҝҳйңҖиҰҒйҮҚж–°еҜ№дҪ“зі»з»“жһ„и®ҫи®ЎиҝӣвҫҸи°ғж•ҙгҖӮ
    1. вҪҗеҰӮ,иҝһй”Ғи¶…еёӮзі»з»ҹдёӯжҲ‘д»¬вҪӨй—®йҡҫжқҘдҝқеӯҳж•°жҚ®,вҪҪеҪ“ж•°жҚ®и¶ҠжқҘи¶ҠеӨҡ,зү№еҲ«жҳҜй”Җе”®и®°еҪ•иҰҒдҝқеӯҳ 1 е№ҙгҖӮиҝҷж—¶еҖҷ,е®ўжҲ·з«Ҝи·ЁвҪ№з»ңеҜ№жңҚеҠЎеҷЁдёҠвҪӮд»¶зҡ„жҹҘжүҫе°ұвҪҗиҫғж…ў,еҸҜиғҪдјҡдҪҝеҫ—жҲ‘д»¬дёҚж»Ўвҫңе®һж—¶жҖ§зҡ„иҰҒжұӮгҖӮ
    2. иҝҷж—¶еҖҷ,еҸҜиғҪжҲ‘д»¬е°ұиҰҒи°ғж•ҙдҪ“зі»з»“жһ„дёӯж•°жҚ®еӯҳеӮЁзҡ„вҪ…жЎҲ,вҪҗеҰӮжҚўжҲҗж•°жҚ®еә“гҖӮжҲ–иҖ…еңЁе®ўжҲ·з«Ҝзҡ„йҖ»иҫ‘еұӮе’Ңж•°жҚ®еұӮд№Ӣй—ҙ,еҠ и®ҫж•°жҚ®жҳ е°„еұӮгҖӮеңЁе®ўжҲ·з«ҜиҝҗвҫҸеҲқе§ӢеҢ–ж—¶,е°Ҷж•°жҚ®д»ҺжңҚеҠЎеҷЁз«ҜиҪҪвјҠеҲ°е®ўжҲ·з«Ҝзҡ„еҶ…еӯҳдёӯеҺ»гҖӮ

дҪ“зі»з»“жһ„йӣҶжҲҗдёҺжөӢиҜ•

  1. еҪ“дҪ“зі»з»“жһ„дёӯеҺҹеһӢеҗ„дёӘжЁЎеқ—зҡ„д»Јз ҒйғҪзј–еҶҷе®ҢжҲҗе№¶з»ҸиҝҮеҚ•е…ғжөӢиҜ•д№ӢеҗҺ,йңҖиҰҒе°ҶжүҖжңүжЁЎеқ—з»„еҗҲиө·жқҘеҪўжҲҗж•ҙдёӘиҪҜд»¶еҺҹеһӢзі»з»ҹ, иҝҷе°ұжҳҜйӣҶжҲҗгҖӮ
  2. йӣҶжҲҗзҡ„зӣ®зҡ„жҳҜдёәдәҶйҖҗжӯҘи®©еҗ„дёӘжЁЎеқ—еҗҲжҲҗдёәвјҖдёӘзі»з»ҹжқҘвјҜдҪң,д»ҺиҖҢйӘҢиҜҒж•ҙдёӘзі»з»ҹзҡ„еҠҹиғҪгҖҒжҖ§иғҪгҖҒеҸҜйқ жҖ§зӯүйңҖ жұӮгҖӮеҜ№дәҺиў«йӣҶжҲҗиө·жқҘзҡ„зі»з»ҹвјҖиҲ¬ дё»иҰҒжҳҜйҖҡиҝҮе…¶жҡҙйңІеҮәжқҘзҡ„жҺҘвјқ,дјӘиЈ…вјҖе®ҡзҡ„еҸӮж•°е’Ңиҫ“вјҠ,иҝӣвҫҸй»‘зӣ’жөӢиҜ•гҖӮ
  3. ж №жҚ®д»ҺжЁЎеқ—д№Ӣй—ҙйӣҶжҲҗзҡ„е…ҲеҗҺйЎәеәҸ,вјҖиҲ¬жңүдёӢеҲ—вјҸз§Қеёёв»…зҡ„йӣҶжҲҗзӯ–з•Ҙ:
    1. еӨ§зҲҶзӮёејҸ(жүҖжңүж”ҫеңЁдёҖиө·пјҢзңӢиғҪдёҚиғҪиҝҮ)
    2. еўһйҮҸејҸ
      1. иҮӘйЎ¶еҗ‘дёӢејҸ
      2. иҮӘеә•еҗ‘дёҠејҸ
      3. дёүжҳҺжІ»ејҸ
      4. жҢҒз»ӯйӣҶжҲҗ
  4. жЁЎеқ—ејҖеҸ‘е®ҢжҲҗеҗҺжҲ‘д»¬иҝӣиЎҢејҖеҸ‘йӣҶжҲҗжөӢиҜ•з”ЁдҫӢ

  2. дјҳзӮ№:еҸҜд»ҘеңЁзҹӯж—¶й—ҙеҶ…иҝ…йҖҹе®ҢжҲҗйӣҶжҲҗ
  3. зјәзӮ№пјҡдёҖж¬ЎиҝҗиЎҢжҲҗеҠҹеҸҜиғҪжҖ§дёҚй«ҳпјҢдёҚе®№жҳ“еҸ‘зҺ°bug

  1. иҮӘйЎ¶еҗ‘дёӢйӣҶжҲҗжҳҜеҜ№еҲҶеұӮж¬Ўзҡ„жһ¶жһ„пјҢе…ҲйӣҶжҲҗе’ҢжөӢиҜ•дёҠеұӮзҡ„жөӢжЁЎеқ—пјҢдёӢеұӮзҡ„жЁЎеқ—з”ЁдјӘиЈ…зҡ„е…·дҪ“жҺҘеҸЈзҡ„жЎ©зЁӢеәҸгҖӮ
  2. иҮӘйЎ¶еҗ‘дёӢйӣҶжҲҗзҡ„дјҳзӮ№:

  2. е°Ҫж—©йӣҶжҲҗжҳҜжҢҮдёҚйңҖиҰҒжҖ»жҳҜзӯүеҫ…вјҖдёӘжЁЎеқ—ејҖеҸ‘е®ҢжҲҗжүҚжҠҠе®ғйӣҶжҲҗиө·жқҘ,вҪҪжҳҜеңЁејҖеҸ‘д№ӢеҲқе°ұеҲ©вҪӨ Stub йӣҶжҲҗиө·жқҘгҖӮ
