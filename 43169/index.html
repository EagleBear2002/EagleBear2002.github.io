Arrays in TLA+

Syntax

The TLA+ syntax for an array expression:

[variable∈set↦expression]

For example:

sqr=[i∈1..42↦i2]

Defines sqr to be an array with index set 1..42 such that sqr[i]=i2 for all i in 1..42.
