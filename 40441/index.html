2021Fall-马克思主义基本原理概论

为什么学习哲学

解决生活问题的需要。在生活中，人们会遇到很多问题，引发我们的思考。实践和行动的前提是理解，而理解就需要学习。

学习哲学是一种生活的需要——理解。理解自我，理解生活，理解我们生活于其中的这个世界。哲学在加深我们的理解方面能够提供有效的帮助。

冯友兰讲，哲学是对人生有系统的反思，这种反思是对自我、对生活和世界深入理解的结果。学习哲学就可以帮我们加深理解。

学习哲学有助于我们健全自我意识（克服精神空虚）、树立完整的世界观、掌握科学的方法论。

什么是哲学

哲学就是关于人生的系统的反思（冯友兰）

哲学就是爱智慧（毕达哥拉斯）

我们应该怎样生活（苏格拉底）

哲学是对宇宙和人生做综合的、辩证的理解。

哲学基本问题的含义

思维与存在的关系问题是哲学的基本问题（思存关系），是恩格斯在《路德维希.费尔巴哈与德国古典哲学的终结》中的界定。

第一，思维和存在何者具有第一性。也就是哪个因素更根本，更具有决定性。唯物主义者的回答是存在第一性，思维第二性。

第二，思维能否认识存在。唯物主义者认为思维能够认识存在，唯心主义者有部分认为可以认识，也有认为不能认识，或者不能完全认识。

试析马克思理解的时间概念及哲学意义

马克思理解的时间概念

感性时间

马克思认为，存在着两种不同时间概念的对立：永恒的时间和感性的时间（伊壁鸠鲁：感觉是判断伦理的标准），马克思实际上是站在感性时间的立场上来反对永恒时间。

马克思认为，时间和感性之间构成了密切的关系，“人的感性就是形体化的时间，就是感性世界的存在着的自身反映”，“事物的时间性和事物对感官的显现，被设定为事物本身的同一个东西。”

劳动时间

在《1844 年经济学-哲学手稿》中，马克思非常重视对异化劳动的分析，他在分析中所揭示的劳动异化实际上也是时间的异化。他通过研究舒尔茨和其他经济学家的著作注意到，先进的、节省劳动的生产技术的采用并没有节省工人的劳动时间，相反，劳动时间却增加了。

马克思说：“劳动是活的、塑造形象的火；是物的易逝性，物的暂时性，这种易逝性和暂时性表现为这些物通过活的时间而被赋予形式。”

自由时间

马克思：“时间是人的积极存在，它不仅是人的生命的尺度，而且是人的发展的空间。”

马克思说：“社会的自由时间是以通过强制劳动吸收工人的时间为基础的，这样，工人就丧失了精神发展所必需的空间，因为时间就是这种空间。”

马克思说：自由王国只有建立在必然王国的基础上，才能繁荣起来。工作日的缩短是根本条件。

节约时间

马克思说：“节约劳动时间等于增加自由时间，即增加使个人得到充分发展的时间。”

“一切节约归根到底都是时间的节约。”

新时间观的意义

  1. 他不仅用时间概念解释感性和现象世界，而且用之分析资本主义社会。时间的视野成为他打开资本主义现实的秘密之关键。
  2. 马克思不仅揭示了时间的积极意义，而且揭示了未来社会实现自由的根本条件。
  3. 社会和大众拥有更多的自由时间，为开启一种新的文明模式创造了新的可能。

试析马克思的存在概念及意义

马克思理解的存在概念

  1. 马克思的“存在”：是各种存在形式的矛盾统一体，包括物质性存在、活动性存在、关系性存在、精神性存在和可能性存在等。
  2. 马克思的存在概念主要是社会性存在，它是新唯物主义的物质概念的延伸和发展。
  3. 马克思扬弃了旧唯物主义物质概念的实体性和机械性，也扬弃了唯心主义实体观的抽象性。
  4. 马克思考察了各种存在形式：自然、人、商品、资本、生产关系、思想、意识形态、宗教、艺术等。
  5. 各种存在形式之间相互作用、相互联系：生产力和生产关系；经济基础和上层建筑；社会存在和社会意识等。
  6. 历史唯物主义从具体的人出发，例如，处在某个特定社会关系下的劳动者。不过多讨论理想化的人。
  7. 《德意志意识形态》：“意识在任何时候都只能是被意识到了的存在，而人们的存在就是他们的实际生活过程。如果在全部意识形态中人们和他们的关系就像在照相机中一样是倒现着的，那么这种现象也是从人们生活的历史过程中产生的”。
  8. 马克思认为，存在主要表现为现实生活的生产和再生产，它在人类历史中起基础作用。
  9. 马克思始终站在现实历史的基础上，不是从观念出发解释实践，而是从物质实践出发解释观念的东西。

对存在概念新理解的当代意义

  1. 马克思突破了传统形而上学对存在的狭隘理解，对存在形式的理解更加广泛，扩展了认识的视野。
  2. 马克思批判了对存在的神秘理解，把存在主要理解为人们的现实生活的生产和再生产，建立了科学的存在论。
  3. 马克思从辩证的角度理解存在，把各种存在形式理解为相互作用、相互关联的统一体。
  4. 坚持从实践出发理解现实的存在，有助于形成实事求是的、完整的世界观。
  5. 马克思对唯心主义观念论进行批判，把存在拉向以物质生产实践为基础的现实生活世界，避免了过度拔高精神的地位，有助于科学地解释世界和改造世界。

试析马克思理解的财富的基础

现今财富的基础

马克思：“现今财富的基础是盗窃他人的劳动时间。”

马克思说：“社会的自由时间是以通过强制劳动吸收工人的时间为基础的，这样，工人就丧失了精神发展所必需的空间，因为时间就是这种空间。”

现实财富的基础

马克思说：“……随着大工业的发展，现实财富的创造较少地取决于劳动时间和已耗费的劳动量，较多地取决于在劳动时间内所运用的作用物的力量，而这种作用物自身——它们的巨大效率——又和生产它们所花费的直接劳动时间不成比例……”

试析马克思理解的真正的财富

“那时，一方面，社会的个人的需要将成为必要劳动时间的尺度，另一方面，社会生产力的发展将如此迅速，以致尽管生产将以所有的人富裕为目的，所有的人的可以自由支配的时间还是会增加。因为真正的财富就是所有个人的发达的生产力。那时，财富的尺度决不再是劳动时间，而是可以自由支配的时间。”

马克思说：“在这个转变中，表现为生产和财富的宏大基石的，既不是人本身完成的直接劳动，也不是人从事劳动的时间，而是对人本身的一般生产力的占有，……总之，是社会个人的发展。

试析马克思关于“资本主义生产的真正限制是资本自身”的论断

马克思：“资本主义生产的真正限制是资本自身，……**生产只是为资本而生产，而不是反过来。生产资料只是生产者社会的生活过程不断扩大的手段。**以广大生产者群众的被剥夺和贫穷化为基础的资本价值的保存和增殖，只能在一定的限制以内运动，……手段——社会生产力的无条件的发展——不断地和现有资本的增殖这个有限的目的发生冲突。”

试析新唯物主义历史观的主要内涵及意义

唯物史观的基本观点

  1. 关注在具体社会关系中的个人而不是理想化的个人。
  2. 一切历史的前提是有生命的个体的存在，要保持肉体的存在，第一个历史活动就是生产物质生活资料。
  3. 人们在从事物质生产的同时，也在创造出他们的宗教、哲学、艺术等精神文化的东西。
  4. 动物的生产是片面的，人的生产是全面的。
  5. 在物质生产的同时，人们又产生一些新的需求，同时产生满足这些新需求的生产活动。
  6. 对历史的理解：历史不仅仅是个体的经验。
  7. 从物质生活出发，就可以理解建立在物质生产基础上的各种社会关系的发展以及竖立其上的上层建筑。
  8. 生产力和生产关系，经济基础和上层建筑，社会存在和社会意识之间的矛盾运动。

对历史的新理解的当代意义

由马克思所建立的这种关于世界和历史的新的理解具有深远的哲学意义。它首先意味着开启了一个崭新的哲学时代，即从唯心主义一统天下到以唯物主义为主流的时代。唯物主义的胜利是和唯心主义的衰落一起发生的，同时也是和形而上学的衰落一起发生的。

其次，新哲学建构起一个崭新的世界观，人们第一次可以用科学完整地解释自然和社会历史的发展。在马克思之前，人们要么用“神意”解释世界历史，要么用理性或者精神解释世界历史，无论哪一种方式，都显示出对自然和社会理解的局限性。根本原因在于，它们都未能把客观的物质世界和人的实践活动作为解释世界历史的基本前提，而是执着于所谓超自然的神圣力量或者崇高的“精神”力量。

第三，由这种新哲学所形成的历史观，第一次从理论上证明了人民群众是历史的创造者，颠覆了传统的以贵族或有产者阶级的统治而形成的狭隘的价值观，赋予了人民以历史发展的主体地位。

马克思实践观的内涵

  1. 物质生产实践（包括劳动工具的生产等）——基础层面（和亚里士多德意义上的创制活动和康德意义上的技术的实践大致重合）
  2. 人自身的生产（劳动者和全面发展的人的塑造）
  3. 关系的生产（社会生产关系、经济关系、交换关系、交往关系、血缘关系、法律关系、政治关系等）
  4. 政治斗争、道德实践等——它包括改造社会的实践，改造的生产关系和政治制度的实践；它是批判的和革命的实践（如争取劳动阶级利益的斗争，争取平等的立法、行政和司法的斗争，劳动者的联合和工农联盟，新道德和克服利己主义的实践等）
  5. 精神生产（包括哲学、道德、经济、法律、文艺、管理等方面的思想的生产，意识形态的生产等）

唯物辩证法的基本主张

  1. 恩格斯：“一个伟大的基本思想，即认为世界不是既成事物的集合体，而是过程事物的集合体，其中各个似乎稳定的事物同它们在我们头脑中的思想映像即概念一样都处在生成和灭亡的不断变化中。”
  2. 发展是在更高基础上的重复（否定之否定），是螺旋式上升，有时是飞跃式的发展；量变转化为质变；对立和统一；唯物主义哲学是一种辩证哲学、过程哲学，是辩证发展学说。
  3. 唯物主义辩证法认为，对事物发生作用的各种力量和趋势所造成的矛盾和冲突推动着事物的发展。
  4. 唯物辩证法认为，每种现象的一切方面相互依存，及其密切而不可分割地联系在一起，这种联系形成统一的、有规律的世界的运动过程。

唯物辩证法的特点

  1. 辩证法的本质是批判的和革命的。
  2. 辩证法在对现存事物进行肯定的理解中包含了对现存事物否定的理解。
  3. 避免极端思维、静止思维。

唯物辩证法的定义

列宁：“辩证法就是关于外部世界和人类思维的运动的一般规律的科学。”
